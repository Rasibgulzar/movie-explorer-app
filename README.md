# Movie Explorer - React Native App

## Overview
A mobile application for browsing movies with authentication, local storage, and API integration.

## Features
- User authentication (Login/Signup)
- Browse movies from API
- View movie details
- Save favorites to local storage
- Settings and notifications
- Responsive design

## Installation
1. Clone repository: `git clone https://github.com/yourusername/movie-explorer-app.git`
2. Install dependencies: `npm install`
3. Run app: `npm start`

## Screens
- Login Screen
- Signup Screen
- Home Screen
- Detail Screen
- Settings Screen

## Technologies
- React Native
- Expo
- React Navigation
- Async Storage
- Expo Notifications
